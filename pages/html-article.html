General

HTML stands for Hyper Text Markup Language and defines the layout and the elements of a webpage. A HTML page must begin with a <!DOCTYPE> tag. It defines the HTML version used in this document.
For HTML 5 it would be <!DOCTYPE html>

Every html document contains the root element <html>, which has a start tag <html> and an end tag </html>. Basically every HTML element is everything from the start tag (<tagname>) to the end tag (</tagname>). HTML elements can also be nested, so that a HTML element can contain multiple elements inside it.

Attributes provide additional information about a html element or are even necessary for void elements like <meta> (name; charset) or <img> (src). Attributes are always defined in the start tag and are written as following: name="value".

Links

HTML links are hyperlinks. You can click on a link and jump to another document.

The HTML <a> tag defines a hyperlink. It has the following syntax:
<a href="url">link text</a>

The <target> attribute specifies where to open the linked document. The <target> attribute can have one of the following values:

  • _self - Default. Opens the document in the same window/tab as it was clicked
  • _blank - Opens the document in a new window or tab
  • _parent - Opens the document in the parent frame
  • _top - Opens the document in the full body of the window

Lists

There are 3 kinds of lists:

Kind of list Tag and meaning Examples
Unordered list <ul>: The list items (<li>) are not ordered (bullet points)

List with list-style:disc

  • Item 1
  • Item 2

List with list-style:circle

  • Item 1
  • Item 2

List with list-style:square

  • Item 1
  • Item 2
Ordered list <ol>: The list items (<li>) are ordered with numbers

List with list-style-type:decimal (default)

  1. Item 1
  2. Item 2

List with list-style-type:lower-alpha

  1. Item 1
  2. Item 2
Description list <dl>: Description list which contains terms and descriptions
Term (<dt>)
Description (<dd>)
Description list
Is something people use sometimes.

You can style the bullet points of a list with list-style-type and can set the position of the bullet points with list-style-position.

Tables

Here is the general structure of a table:

Table Header 1 Table Header 2 Table Header 3
Table cell 1 Table cell 2 Table cell 3
Table footer cell 1 Table footer cell 2 Table footer cell 3

Source code:

                    <table>
                          <thead>
                                <tr>
                                      <th>Table Header 1</th>
                                      <th>Table Header 2</th>
                                      <th>Table Header 3</th>
                                </tr>
                          </thead>
                          <tbody>
                                <tr>
                                      <td>Table cell 1</td>
                                      <td>Table cell 2</td>
                                      <td>Table cell 3</td>
                                </tr>
                          </tbody>
                          <tfoot>
                                <tr>
                                      <td>Table footer cell 1</td>
                                      <td>Table footer cell 2</td>
                                      <td>Table footer cell 3</td>
                                </tr>
                          </tfoot>
                    </table>

To add a solid border around the cells you have to a border to the th and hd elements. The border property has following syntax: border: <size> <type> <color>.
A solid black border that is 1 px thick would be: border: 1px solid black.

If the borders from the cells overlap you can use the border-collapse property to collapse the borders into a single border.
