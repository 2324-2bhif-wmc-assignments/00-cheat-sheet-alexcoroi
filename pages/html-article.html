General

HTML stands for Hyper Text Markup Language and defines the layout and the elements of a webpage. A HTML page must begin with a <;!DOCTYPE>; tag. It defines the HTML version used in this document.
For HTML 5 it would be <;!DOCTYPE html>;

Every html document contains the root element <;html>;, which has a start tag <;html>; and an end tag <;/html>;. Basically every HTML element is everything from the start tag (<;tagname>;) to the end tag (<;/tagname>;). HTML elements can also be nested, so that a HTML element can contain multiple elements inside it.

Attributes provide additional information about a html element or are even necessary for void elements like <;meta>; (name; charset) or <;img> (src). Attributes are always defined in the start tag and are written as following: name="value".

Links

HTML links are hyperlinks. You can click on a link and jump to another document.

The HTML <a> tag defines a hyperlink. It has the following syntax:
<a href="url">link text</a>

The <target> attribute specifies where to open the linked document. The <target> attribute can have one of the following values:

  • _self - Default. Opens the document in the same window/tab as it was clicked
  • _blank - Opens the document in a new window or tab
  • _parent - Opens the document in the parent frame
  • _top - Opens the document in the full body of the window
